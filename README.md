# Ads Metrics Calculator

Утилита для расчета эффективности рекламных кампаний (ROI, CR, Net Profit). 

## Структура проекта
- `app/` - запускаемое CLI-приложение (точка входа).
- `packages/core/` - переиспользуемая бизнес-логика.
- `tests/` - Smoke и Unit тесты (включая граничные случаи).
- `docs/` - архитектурное описание и диаграммы.

## Быстрый старт
Для автоматизации рутинных действий используется `Makefile`.

- **Запустить тесты**: `make test`
- **Запустить приложение**: `make run`
- **Полная проверка (тесты + запуск)**: `make check`
