# Ads Metrics Calculator

**Ads Metrics Calculator** — это быстрый и воспроизводимый CLI-инструмент (интерфейс командной строки), разработанный для медиабайеров, арбитражников и дата-аналитиков. Он обрабатывает сырые данные о трафике из различных партнерских сетей и мгновенно рассчитывает ключевые показатели эффективности (KPI), такие как ROI, Конверсия (CR) и Чистая прибыль (Net Profit).

### 🐳 Быстрый запуск через Docker
Приложение полностью контейнеризировано! Вы можете запустить его локально без установки Python или настройки окружения:
```bash
make compose-up
```
*(Никакой возни с базами данных или конфликтами зависимостей)*

## Какую проблему мы решаем
Работа с данными о трафике из множества партнерских сетей (например, Leadgid, Mitgo, Shakes.pro) часто сводится к работе с сырыми JSON-выгрузками. Расчет метрик вручную в Excel отнимает много времени и чреват ошибками. **Ads Metrics Calculator** решает эту проблему путем жесткого разделения логики парсинга сырых данных и чистой математической бизнес-логики. Это гарантирует безопасность расчетов и корректную обработку граничных случаев, таких как нулевые клики или отрицательные затраты.

## Ключевые особенности
* **Строгие архитектурные границы:** Бизнес-логика полностью изолирована от CLI-интерфейса, что соответствует лучшим практикам программной инженерии.
* **Парсинг JSON:** Автоматическое чтение и обработка стандартных дампов трафика.
* **Защита от граничных случаев:** Встроенные предохранители против деления на ноль и отрицательных финансовых значений.
* **Автоматизированное тестирование:** Исчерпывающее покрытие тестами, включая `smoke` и `unit` тесты (с позитивными и негативными сценариями) с использованием встроенной библиотеки `unittest`.
* **Воспроизводимое окружение:** Полностью докеризированная среда выполнения с использованием `compose.yaml`.
* **Автоматизация рабочих процессов:** Единый интерфейс `Makefile` для сборки, тестирования и запуска проекта.

## Стек технологий
* **Бэкенд и логика:** Python 3.12 (Только стандартная библиотека, без сторонних зависимостей)
* **Тестирование:** `unittest`, `subprocess`
* **Инфраструктура:** Docker, Docker Compose
* **Автоматизация:** GNU Make

## Как работают метрики
Мы не просто вслепую делим числа. Наш основной модуль (`packages/core/metrics.py`) применяет строгие математические правила:
* **ROI (Окупаемость инвестиций):** Расчет рентабельности по формуле `(доход - расход) / расход * 100`. *Статус: Выбрасывает `ValueError`, если расходы отрицательные, чтобы предотвратить ложные финансовые отчеты.*
* **CR (Коэффициент конверсии):** Оценка эффективности генерации лидов по формуле `(лиды / клики) * 100`. *Статус: Безопасно возвращает `0.0`, если клики равны 0, предотвращая падение приложения.*
* **Net Profit (Чистая прибыль):** Классический расчет `доход - расход` для мгновенной оценки финансового результата.

## Локальная установка и запуск

Чтобы запустить Ads Metrics Calculator на вашем компьютере, выполните следующие шаги:

**1. Клонируйте репозиторий**
```bash
git clone [https://github.com/ВАШ_НИКНЕЙМ_НА_GITHUB/ads-metrics-calculator.git](https://github.com/ВАШ_НИКНЕЙМ_НА_GITHUB/ads-metrics-calculator.git)
cd ads-metrics-calculator
```

**2. Проверьте проект (Запуск тестов)**
Убедитесь, что всё работает корректно на вашей системе, запустив набор автотестов:
```bash
make test
```

**3. Запустите приложение локально (Нативно)**
Запустите скрипт, используя предоставленный тестовый файл с данными о трафике:
```bash
make run
```

**4. Запустите через Docker Compose (Рекомендуется)**
Выполните программу в полностью изолированном контейнере:
```bash
make compose-up
```
