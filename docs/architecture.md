# Архитектура системы

Проект разделен на два основных слоя для обеспечения тестируемости и переиспользования:

1. **Application Layer (`app/cli/`)**: Запускаемая часть. Отвечает за чтение JSON, оркестрацию вызовов и вывод результатов в консоль.
2. **Core Layer (`packages/core/`)**: Переиспользуемое доменное ядро. Содержит чистую бизнес-логику (расчет ROI, CR, Profit). Полностью изолировано от ввода-вывода и покрыто unit-тестами.
