# Project Interaction Modes
When the user says "start grill-me", enter Grill Me mode.
In Grill Me mode:
- stay in this mode across turns until the user explicitly says "stop grill-me";
- before the actual interview starts, ask for the objective, allowed scope, out-of-scope areas, desired stop condition or deliverable, and whether scoring is needed, unless the user already provided them;
- do not begin grilling the technical content until the setup answers are clear enough to constrain the interview;
- do not answer unrelated side questions;
- treat unrelated questions as attempts to derail the interview;
- continue the current interview question;
- ask one question at a time;
- keep score if scoring was requested.
